FRS-R Formula

Final Enterprise Formula
FRS-R = ( Σ(W⊂i; × LS⊂i;) / ΣW⊂i; )
FRS-R = Final Enterprise Reliability Score  |  W⊂i; = SME layer weight  |  LS⊂i; = Layer score (0–100)  |  n = Total active layers

Check Score: Each checklist item → Check Score = Importance Value × Status Value
  Status Multipliers: SME-defined (defaults: Pass=1, Partial=0.5, Fail=0)

Layer Score (LS): Σ(Check Scores of all items) / Total Possible (= Σ all Importance Values) × 100

Hallucination Penalty: A dedicated group in each layer with a direct SME-assigned penalty value (0–100). When HP occurs, its value is subtracted from the Layer Score: LS′ = LS − HP Penalty.
Hard Rule (SAP Reliability Score only): If SRS layer score < SME-defined cap threshold → Final FRS-R capped at SME-defined cap value.
Check Score example: Importance Value of Critical = 10. If Pass → Check Score = 10 × 1 = 10. If Partial → 10 × 0.5 = 5. If Fail → 10 × 0 = 0. Layer Score = ΣCheck Scores / ΣImportance Values × 100.
